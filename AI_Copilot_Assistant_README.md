# 🤖 AI Copilot Assistant

AI Copilot Assistant is a Flask-based AI coding assistant powered by the OpenAI API. It provides real-time coding help, explanations, and productivity assistance through an interactive chatbot interface.

The project demonstrates how generative AI can be integrated into web applications to create a Copilot-style developer assistant similar to modern AI programming tools.

---

## 🚀 Features

- 💬 AI-powered coding assistant
- 🧠 Programming question answering
- 📘 Code explanation support
- ⚡ Real-time chatbot responses
- 🌐 Flask web application
- 🔑 OpenAI API integration
- 🎨 Simple and clean user interface
- 🚀 Easy deployment and setup

## ⚙️ Installation

### Prerequisites
- Python 3.9+
- OpenAI API Key

### Setup Steps

1. Clone the repository:
```bash
git clone https://github.com/amrutchikkamath/AI-Copilot-Assistant-OpenAI-API-Python-Flask-.git
```

2. Navigate to project folder:
```bash
cd AI-Copilot-Assistant-OpenAI-API-Python-Flask-
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Add your OpenAI API key inside the project.

5. Run the Flask application:
```bash
python app.py
```

6. Open browser:
```text
http://127.0.0.1:5000
```
