# AI Copilot Assistant

This is a Copilot-style AI assistant built using OpenAI API, Python, and Flask.

## Features
- Ask coding questions
- Get explanations
- Productivity assistant
- Copilot-style responses

## Setup

1. Install requirements:
pip install -r requirements.txt

2. Add your OpenAI API key in .env file

3. Run:
python app.py

4. Open browser:
http://127.0.0.1:5000
